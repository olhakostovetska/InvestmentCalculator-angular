# 💰 Investment Calculator (Angular)

A simple investment calculator built with Angular.  
This app allows users to calculate annual investment growth based on initial investment, annual contribution, expected return, and duration.

---

## 🚀 Features

- 📥 User input form (two-way binding with `ngModel`)
- 🔁 Real-time calculation on form submission
- 📊 Dynamic results table
- 💸 Currency formatting using Angular `CurrencyPipe`
- 🔗 Component communication via `@Input` and `@Output`
- ⚡ Built with Angular standalone components

---

## 🛠️ Technologies

- Angular (Standalone Components)
- TypeScript
- HTML / CSS
- Angular Forms (`FormsModule`)
- Angular Pipes (`CurrencyPipe`)
